Accessibility Statement

Last updated: January 2026

Our commitment

Inter Active Move B.V. strives to make this website accessible to everyone, including people with disabilities. We believe that the web should be available to everyone, regardless of physical or cognitive limitations.

Accessibility standard

We aim for conformity with the Web Content Accessibility Guidelines (WCAG) 2.1 level AA. These guidelines help make web content more accessible to people with various disabilities, including:

  • Visual impairments (blindness, low vision, color blindness)
  • Hearing impairments (deafness, hard of hearing)
  • Motor impairments
  • Cognitive impairments

Measures we have taken

  • Keyboard navigation: The website is fully navigable with the keyboard
  • Skip link: A "Skip to content" link is available for screen readers
  • Language attribute: Pages have the correct language attribute (nl/en)
  • Alt texts: Images have descriptive alternative texts
  • Contrast: We strive for sufficient color contrast
  • Forms: Form fields have clear labels
  • Responsive design: The website works on various screen sizes
  • Reduced motion: Animations respect the "prefers-reduced-motion" setting

Known limitations

Despite our efforts, some parts of the website may not yet be fully accessible:

  • Some older images may lack detailed alt texts
  • The interactive canvas background is decorative and not accessible, but contains no essential information
  • Videos currently do not include subtitles

We are continuously working on improving accessibility.

Compatibility

This website is designed to be compatible with:

  • Modern browsers (Chrome, Firefox, Safari, Edge)
  • Screen readers (NVDA, JAWS, VoiceOver)
  • Mobile devices and tablets
